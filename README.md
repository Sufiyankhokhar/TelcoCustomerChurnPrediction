# Teclo Customer Churn Prediction

# Description
This project predicts whether a telecom customer will churn (leave the service) or not
using Machine Learning techniques. The project is built using Python and deployed using
a Streamlit web application.

# Problem Statement
Customer churn is a major challenge in the telecom industry.
The objective of this project is to identify customers who are likely to churn
so that proactive retention strategies can be applied.

# Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn

# ML Model Used
Logistic Regression
Random Forest Classifier

# Model Evaluation Matrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score
- Confusion Matrix

# Features Used
- Tenure
- Monthly Charges
- Total Charges
- Contract Type
- Internet Service
- Payment Method

# How to Run the Project
1. Clone the repository:
```bash
git clone https://github.com/Sufiyankhokhar/TelcoCustomerChurnPrediction.git
```
2. Install required libraries: pip install -r requirements.txt

3. Run the Streamlit app: streamlit run app.py
